• Install Git
  • Check in terminal => git --version
  • Git init => to init the git in to a project
  • Git add => single or all files
  • Check => Git Status
  • Git commit -m "your message here"
  • Git log => check the changes with message is displayed
  • Git reset =>
    • git reset --hard enter the commit code here. Whole code changed to previous version
    • git reset --soft enter the commit code here. Git log is deleted
  • Git origin => git remote add origin address in github. where is to be pushed
  • Git branch => git branch -M main
  • Git push => git push -u origin main
  • Git clone => clone & use others project
  • Git pull=> pull any changes made in others project
  • Git branch => Shows the branchs of git. to add any branches git branch enter_branch_name
  • Git checkout => to move one branch to other. git checkout enter_branch_name
  • Git merge => git merge enter_the_branch name. after your code is done. then merge code to main branch.
